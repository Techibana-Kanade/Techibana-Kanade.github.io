函数构成

函数原型 例int sum(int a,int b); 它的作用是告诉编译器有一个名叫sum的函数，返回值类型是int,参数是int ，int，注：参数只需要给出类型即可。另外，在函数定义中，只有参数的类型是必要的。例如int sum(int,int)。
即

#include<stdio.h>
int imin(int x,int y);

int main(void){
int a,b,c;
scanf(“%d %d”,&a,&b);
printf(“%d和%d中最小的数是：%d”,a,b,imin(a,b));

}
int imin(int x,int y){
int min=0;
if(x>y){
min=y;
}else{
min=x;
}
return min;
}

关键字`return`后面的表达式的值就是该函数的返回值。  
该代码最后一句`return min`中变量`min`的值就是函数的返回值。因为`min`是int型的变量，所以`imin()`的函数类型是int。    
return语句把`min`的值传回给了主函数(而不是把min和值都传回给主函数，因为变量`min`是函数imin()的局部变量，主函数不知道`min`的存在)。故需要主函数中的一个变量去接收这个值。  
所以应当把返回的值赋给主函数的一个变量。演示代码没有这么做。给出操作方法，即`int less=imin(a,b);`。

以下是文言一心的解释：  
在 C 语言中，函数参数是通过值传递的，也就是说，当你调用一个函数时，实际上是将参数的值复制一份传递给函数，而不是传递参数本身。

在你给出的代码中，`imin(a, b)` 调用会把 `a` 和 `b` 的值复制一份传递给 `imin` 函数。在 `imin` 函数内部，这些复制的值被用来进行比较，然后返回最小值。这个返回的最小值是通过 `return` 语句返回的，它会被赋给调用函数 `main` 中的 `c` 变量。  
return 语句是在 imin 函数内部执行的，它会将函数的结果返回给调用函数，也就是 main 函数。然后，这个结果会被赋给 c 变量。所以，当你执行 c = imin(a, b); 时，imin(a, b) 的返回值会被赋给 c。

因此，当你调用 `c = imin(a, b);` 时，你实际上是将 `a` 和 `b` 的值复制给 `imin` 函数，然后使用 `imin` 函数返回的最小值来设置 `c`。

"函数调用imin(a,b)只是把两个变量的值拷贝了一份"这句话的意思是，当你调用函数时，你传递的是变量的值，而不是变量本身。因此，在函数内部，你实际上是在操作这些值的复制品，而不是原始变量。这就是为什么你可以在函数内部改变参数的值，但不会影响原始变量的值。
