第二周变量定义1.变量:用于存储数据的存储空间 2.变量的定义：告诉编译器创建一个指定类型和名称的变量 <类型名称><变量名称>例如： int age; 定义了一个名为age变量，类型为int 3.变量的名字是一种标识符，只能由数字、下划线、字母组成。但是数字不能再标识符的开头。 C语言关键字不可用做标识符.如 int 1age; 错误,为数字开头、int int 错误，关键字不可做标识符。 4.变量赋值；将一个值存储在变量中 如int age; age = 10;5.变量初始化；在创建变量时为其分配一个初始值 如int age =10注：在C语言中，变量可以被视为一个用于存储数据的存储空间。要存储数据，您需要先定义一个变量，即创建一个存储空间，并为其指定一个名称。然后，您可以通过赋值操作符将值分配给变量，从而将其初始化为一个特定的值。如果没有显式地初始化变量，它将包含一个不确定的值，即所谓的“垃圾值”。这是因为在C语言中，变量的初始值是不确定的，直到您显式地为其赋值为止。因此，为了确保您的程序能够正确地运行，最好始终初始化变量。

结构结构是一种可以自定义的数据类型。 结构声明及定义结构变量结构定义描述了一个结构的组织布局。示例如下： 12345struct book{ char title; char author; float value;t}; 这个代码声明了一个由两个char类型变量和一个float变量组成的结构。struct表示跟在其后的是一个结构。book是一个标签，可用此标签来声明使用book结构布局的结构变量变量。如struct book library;表面创建了一个类型为struct book的结构变量library.struct相当于一般声明中的int、float等。也可以不使用标签。如下： 12345struct { char title; char author; float value;}library; 这结构变量的声明和定义可以一步完成。不过最好使用标签来定义，这样方便。
